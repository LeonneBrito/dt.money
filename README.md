# :computer: How to run

```bash
# Clone Repository
$ git clone https://github.com/LeonneBrito/moveit-nlw4.git
```

```bash
# Install Dependencies
$ yarn

# Run Aplication
$ yarn dev
```
Go to http://localhost:3000/ to see the result.
